Function Problem Sets

  1. isEven(): write a function which takes a single numerical argument and remains true if the number is even, and false otherwise.

  2. factorial(): write a // function which takes a single numerical argument and returns the factorial of that number.

  3. kebabToSnake(): write a function which takes a single kebab-based string argument and returns the snake_cased version. (Basically replace "-"s with "_"s.)
